# AECODE Sales Dashboard

## Scope

Dashboard publico y sanitizado construido desde la pestaña `BBDD Ventas General`.

## Rules

- No publicar nombres, correos, telefonos, links de vouchers ni comentarios con datos personales.
- Mantener solo metricas agregadas y registros anonimos por programa/canal/estado.
- Si se necesita detalle nominal, debe vivir en Google Sheets o un entorno autenticado, no en GitHub Pages.
- El dashboard debe responder decisiones comerciales: ventas, cobro, programas, canales, pendientes, anulados y calidad de datos.
- La app debe poder desplegarse como sitio estatico sin backend.
